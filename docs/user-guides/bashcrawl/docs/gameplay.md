# Gameplay Guide

This guide explains the core mechanics of Bashcrawl.

## Navigation

Navigate through the dungeon using the `cd` command. Use `ls` to see the contents of the current room.

## Interacting with Objects

Use `cat` to read scrolls and other files. Other commands will be revealed as you progress.

## Inventory

Your inventory is stored in the `$I` environment variable. You can view it with `echo $I`.

## Health

Your health is stored in the `$HP` environment variable. You can view it with `echo $HP`.
